# Adobe認證常見問題集

若要進一步了解Adobe認證計畫，請參閱下方的常見問題集(FAQ)。

![橫幅](/help/certifications/assets/home_banner_narrow.png)

>[!BEGINTABS]

>[!TAB 一般]

+++什麼是認證？哪些好處？

個人優點：認證是認證個人在Adobe應用程式中的技能和能力的憑據。 通過認證考試的Adobe成為認證專家、專家或碩士。

認證需要承諾、具有法律上的防禦性，以及Adobe您的專業技能的驗證。 您也可以從下載徽章，在您的社交設定檔上展示您的認證 [!DNL Credly].

若要進一步了解如何在 [!DNL Credly],go [此處](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++我如何知道要參加哪項考試？

認證是基於角色的，這意味著有多門考試可供使用。 檢閱認證歷程頁面，以選擇要參加的考試。 通過以下方式選擇考試：

* 應用程式：例如， [!DNL Adobe Analytics] 或 [!DNL Marketo Engage]
* 角色：例如，開發人員或業務從業人員
* 級別：例如，專業人員，0-12個月的經驗；或專家1-3年

有時有多個版本的考試。 最好採用最新版本。

* 如果即將退出考試，您會在認證歷程頁面上看到附註。 這表示此考試版本很快將被刪除，因為有新版本的考試可用。 我們鼓勵候選者選擇最新版本，但如果你更願意接受，你可以選擇退休版本。 無論如何，你的認證將持續兩年。
* 如果您目前的認證即將過期，您應進行更新。 續約選項有較少的問題，且免費。 這些選項涵蓋最新的應用程式功能。 如果您的認證過期，您將必須參加最新版的認證考試，而您需為此付費。 [深入了解續約選項](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++如果我在經過監督的考試中遇到技術問題呢？

PSI和 [!DNL Examity] 通過聊天、電子郵件和電話獲得支援。 Adobe支援無法解決考試期間發生的技術問題。

[!DNL Examity] 支援 - `adobepartnersupport@examity.com`

PSI支援 —  `psisupport@psionline.com`

最快的幫助資源是通過聊天。 所有考生在參加考試前，都應確保電腦具有適當的系統要求。

如需詳細資訊，請參閱：PSI和 [!DNL Examity] 用於管理考試和技術設定的頁面。

+++

+++我可以與誰聯繫以解答認證問題或問題？

有關考試註冊、考試付款、考試交付、測試中心、考試評論、考試問題、測試體驗、測試適應或有關認證門戶的問題，請聯繫 `certif@adobe.com`.

+++

+++Adobe能否容納有特殊要求的候選人？

Adobe可滿足某些特殊需求。 如果您需要更多時間來完成考試或其他住宿，請聯繫 `certif@adobe.com` 和你的要求。

**您的申請必須在考試安排前2週發送。**

+++

+++如果失敗，何時可以再次嘗試認證考試？

若要取得或保持您的Adobe認證狀態，您需要重新參加最新的認證考試。 建議您在認證過期之前，有充足的時間考慮可能的重新使用。

對於第一次嘗試Adobe認證考試不及格的應聘者，必須等待至少24小時再次嘗試。 如果他們在第二次或隨後的任何嘗試中都失敗，在他們可以重考之前，需要14個日曆日的等待期。 每次嘗試都會分別收取考試費。

+++

+++如何取得分數報表的副本？

* 步驟1:登入 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"}、 CertMetrics 。 使用您的ID或電子郵件地址和密碼登入。
* 步驟2:按一下「成績單」。
* 步驟3:滾動查看您的記錄，並選擇要查看的報表。 您可以將報表以電子郵件傳送至您，或在螢幕上檢視。

+++

+++如何追蹤認證狀態？

您可以前往 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 在「認證」標籤下。

+++

+++通過認證考試後能得到什麼？

考試完成後，結果會立即顯示在螢幕上。 在72小時內，您會收到CertMetrics的電子郵件確認函，內含憑證PDF。 您可以從CertMetrics帳戶中存取顯示認證狀態的標誌，並可用於您的網站、電子郵件、簽名、名片、社交個人檔案等。

+++

+++如何登入或建立Adobe憑證管理/特定量度帳戶？

您可以 [在這裡登入或建立帳戶](https://www.certmetrics.com/adobe/){target="_blank"}. 您可在此處查看可用的考試、考試記錄、認證過期日期以及您可獲得的任何優惠，如折扣。

+++

+++認證持續多久？

由於我們需要確保我們經過Adobe認證的個人具備新的應用程式功能，因此Adobe認證是 **有效期為兩年**.

+++

+++除了英語之外，還有其他語言的考試嗎？

通過PSI（我們的供應商之一）提供一些日文考試。 如果考試有日文版，該考試的考試詳細資訊中將注明。 在認證歷程頁面的「準備就緒」標籤上可找到測試詳細資訊。

+++

+++您是否在Adobe活動中提供認證？

某些Adobe贊助的活動會提供認證。

+++

+++認證是否與特定版本的應用程式(例如Adobe Experience Manager 6.5)相關聯？

不是直接的。 但是，由於認證現在需要每2年更新一次，因此您的認證可確保您保持最新版本的Adobe應用程式。

+++

>[!TAB 通過按需考試續訂]

+++什麼是認證續約？ 如何續訂認證？

Adobe認證會在兩年後到期。 您可以參加並通過最新版的適用Adobe認證考試，以續約認證。

您有兩個選項：

* 定期參加認證考試
* 以續約考試為例 — 較短的考試，重點介紹最新的解決方案功能以及您已了解的事項。

如果您沒有原始認證或您的認證已過期，則無法參加續訂考試。

我們偶爾會為某個角色而退休考試。 在這種情況下，您的認證將持續兩年，然後該認證將會過期。 例如，我們不再提供Lead/Dev Ops認證，因此認證將在不進行替代測試的情況下到期。

+++

+++如果我未在認證過期前續約，會發生什麼事？

您的認證將過期，且會顯示為過期於 [!DNL Credly] 和任何社交資料。 此外，如果您的認證過期，您將無法參加較短的續訂考試，或透過持續學習續約。

若要進一步了解如何在 [!DNL Credly],go [此處](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++續訂我的認證需要多少費用？

您可以免費續約認證。 這為您提供了一種經濟實惠的方法，可維護您的認證並保持您的解決方案技能最新。

若要進一步了解如何續訂認證，請前往 [此處](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++哪裡可以看到我的認證到期日？ 我會收到通知嗎？

登入 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"}. 您可以在此處查看憑證何時到期。 您也應檢查您的設定檔，並確認您的連絡人、次要電子郵件和工作資訊正確無誤。

認證過期前一個月，您會收到電子郵件通知。 但如果您已變更工作，且未更新電子郵件，則可能不會收到通知。 在您的CertMetrics帳戶中新增次要電子郵件（例如Gmail、Hotmail、Yahoo等），如此一來，即使您變更公司，您也會收到憑證過期提醒電子郵件。

+++

+++我需要參加哪項考試才能續學？

若要了解您需要接受哪項考試，請登入 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 使用您的Adobe ID，按一下「認證」，然後選取認證名稱。 續訂考試列於 **保持最新的要求**.

+++

+++我續約認證後，會獲得新的數位徽章嗎？

不可以。 您的數位徽章將維持不變。

+++

+++我是否可以參加最初認證時參加的相同考試以續訂認證？

否，一旦通過初始認證，您就無法重考相同的考試續期。

+++

>[!TAB 通過持續學習進行更新]

+++什麼是Adobe的持續學習認證續約計畫？

Adobe提供持續的學習和參與活動，讓候選人能夠保持活躍狀態，並將其認證狀態延長2年。 這僅適用於特定應用程式。

+++

+++我的認證已過期；我可以繼續學習和參與活動以續約嗎？

否，此續約選項僅適用於Adobe經主動認證的Analytics、Experience Manager、商務和Target候選項。

+++

+++哪些認證提供持續學習和參與活動，以便續約？

目前，此計畫包含Analytics、Experience Manager、商務和Target。 我們會透過適用管道分享公告，當我們擴展公告以取得其他認證時。

+++

+++哪些持續學習和參與活動是認證續訂計畫的一部分？

將提供5至7門隨選課程，供考試。 續訂認證的標準是根據課程通過任何3項評估，或通過2項評估，並提交經簽名的客戶參考表。

+++

+++什麼是認證續約的客戶參考表？

候選人可提交的表單，作為Adobe持續學習和參與活動的一部分，以續訂其認證。 該表格必須由客戶簽署，說明候選人在過去12個月內在與其續約認證產品相關的專案上至少花費了100小時。 通過將此選項納入認證續訂要求，我們希望確認您的持續參與是驗證您的技能和知識的一種方式，這樣您的認證就會保持活躍。

+++

+++通過持續學習和參與活動來尋求認證續訂是否需要支付費用？

不可以。 所有活動均可通過Adobe憑據管理器免費訪問。

+++

+++我是否必須通過評估並提交客戶參考表？

否，提交客戶參考表是可選的，而且候選人可以完成其中一項活動，以續訂其認證。 例如， [!DNL Adobe Analytics] 認證續約需要從指定清單中至少完成3個活動。 該清單包括通過6項評估並提交客戶參考表。 候選人可通過2次評估並提交客戶參考表，從而滿足續訂的3項最低要求。

+++

+++我在一個解決方案中擁有三個認證。 我是否需要完成每項認證的要求？

不需要，您只需要完成針對解決方案列出的需求。 完成所需的活動並將結果提交到Adobe憑證管理器後，將續訂您的認證。 例如，若您持有 [!DNL Adobe Analytics] 身為業務從業者、開發人員和架構師，您只需完成3個活動，即可滿足所有要續訂的認證的續約需求。

+++

+++我是否需要檢查課程，還是乾脆開始進行評估？

這些評估以Experience League課程為基礎，審查這些課程將使候選人有能力成功通過。 然而，續期的關鍵要求只是通過評估。

+++

>[!TAB 重新啟動程式]

**計畫概述**

+++什麼是Adobe認證重新啟動計畫？

Adobe認證重新啟動計畫是一次性計畫，允許個人通過線上短期考試或評估，免費重新獲得過期的Adobe認證。

+++

+++重新啟動程式將運行多久？

該方案將於2023年4月24日至10月20日執行六個月。

+++

+++未來Adobe會再次提供這類計畫嗎？

目前，我們不打算再提供此功能。 這只是一次性優惠。

+++

+++重啟計畫包含哪些認證級別？

重新啟動計畫中只包括專家級和主級認證。 從2023年6月起，專業級認證將提供續約途徑。

+++

**計畫資格**

+++誰有資格參加Adobe認證重啟計畫？

從2020年到2023年4月23日，認證過期的個人有資格參加重新啟動計畫，條件是他們尋求的認證包含在計畫中（請參閱以下的合格認證考試清單）。

+++

+++ 為什麼多年來發佈的所有認證考試都沒有納入重啟計畫？\
2019年前發行的考試不包括在「重新開始」計畫中。

+++

+++為什麼2023年4月23日是重新啟動計畫資格的截止日期？

因為它讓無法參加免費認證續約期（2023年1月23日推出）的個人有足夠的時間續約其認證。

+++

+++哪些認證屬於Adobe認證重啟計畫？

除已淘汰的認證外，所有使用中的Adobe認證都屬於計畫的一部分。

+++我的認證將在2023年4月24日到10月20日之間過期，而此計畫可用。一旦過期，我可以繼續重新啟動認證計畫嗎？

不可以。 自推出之日起及之後，所有持有活躍認證的候選人都可選擇繼續我們於2023年1月宣佈的持續學習更新路徑。

重啟計畫僅針對在2023年4月24日之前過期的認證，以便為無法參加連續學習計畫的候選人提供認證。

**[!DNL Marketo Engage]解決方案架構師(MCSA)和Adobe Commerce**

+++我已過期 [!DNL Marketo] 認證解決方案架構師(MCSA)。 我可以通過本計畫重新聲明嗎？

已過期的候選項 [!DNL Marketo] 經認證的解決方案架構師認證可以尋求Marketo業務從業人員重啟路徑，以滿足對新 [!DNL Marketo Engage] 架構師認證。

最後要求是通過新的 [!DNL Marketo Engage] 建築師考試。 對於所有MCSA過期的證書持有者，在2023年12月31日之前，我們將進行50%的折扣促銷，請聯繫認證支援以了解詳細資訊。

+++

+++為什麼Adobe [!DNL Commerce] 認證不屬於Adobe認證重新啟動計畫嗎？

當前提供的Adobe沒有過期的證書持有者 [!DNL Commerce] 自2023年4月23日起的認證。 因此，所有活動Adobe [!DNL Commerce] 經過認證的候選人可以選擇通過持續學習續訂 [認證的續訂路徑](https://experienceleague.corp.adobe.com/docs/certification/certification/technical-certifications/ac/ac-renew.html?lang=en){target="_blank"}.

+++

**Adobe憑據管理器配置檔案**

+++我為什麼看到 _參與率_ 在「Adobe憑據管理器」的「配置檔案」的「歷史記錄」部分下？

這是為了標籤您是否符合Adobe認證重啟計畫的資格。 已分配此參與的候選者可以完成要求，以在每個應用程式中重新啟動其過期的認證。
