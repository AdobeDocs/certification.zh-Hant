# Adobe認證常見問題

要瞭解有關Adobe認證計畫的更多資訊，請參閱以下常見問題(FAQ)。

![橫幅](/help/certifications/assets/home_banner_narrow.png)

>[!BEGINTABS]

>[!TAB 一般]

+++什麼是認證，有哪些好處？

個人好處：認證是一種證書，用於識別個人在Adobe應用程式中的技能和能力。 通過認證考試的候選人將成為Adobe認證專家、專家或碩士。

認證需要承諾、具有法律上的防禦性，並且需要Adobe您的專業知識來驗證。 您還可以通過從下載您的徽章，在您的社交個人資料中顯示您的證書 [!DNL Credly]。

瞭解有關如何管理徽章的詳細資訊 [!DNL Credly]按鈕 [這裡](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}。

+++

+++如何知道要參加哪項考試？

認證是基於角色的，這意味著有多個可用的考試。 查看認證行程頁面以選擇要參加的考試。 您將通過以下方式選擇考試：

* 應用程式：比如說， [!DNL Adobe Analytics] 或 [!DNL Marketo Engage]
* 角色：例如，開發人員或業務從業人員
* 級別：例如，專業，0-12個月的經驗；或專家1-3年

有時考試有多個版本。 最好採用最新版本。

* 如果要退休，您將在認證行程頁面上看到一個注釋。 這意味著此考試版本很快將被刪除，因為有新版本的考試可用。 我們鼓勵候選人選擇最新版本，但如果你更願意接受，你可以選擇退休版。 無論如何，你的認證將持續兩年。
* 如果您當前的證書即將過期，您應續訂。 續訂選項的問題較少，並且是免費的。 這些選項涵蓋最新的應用程式功能。 如果您的認證過期，您必須參加最新版本的認證考試，您需要為此付費。 [瞭解有關續訂選項的詳細資訊](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}。

+++

+++如果在我的輔導考試期間有技術問題怎麼辦？

PSI和 [!DNL Examity] 通過聊天、電子郵件和電話獲得支援。 Adobe支援無法幫助解決考試期間出現的技術問題。

[!DNL Examity] 支援 - `adobepartnersupport@examity.com`

PSI支援 —  `psisupport@psionline.com`

最快的幫助資源是通過聊天。 所有test應試者在參加考試前應確保電腦具備適當的系統要求。

有關詳細資訊，請參閱：PSI和 [!DNL Examity] 用於管理考試和技術設定的頁面。

+++

+++我可以與誰聯繫，瞭解認證問題或問題？

有關考試註冊、考試付款、test交付、測試中心、考試評論、考試問題、測試經驗、test適應或認證門戶問題的問題，請聯繫 `certif@adobe.com`。

+++

+++Adobe能容納有特殊要求的候選人嗎？

Adobe可以滿足某些特殊要求。 如果您需要更多時間完成考試或其他住宿，請聯繫 `certif@adobe.com` 你的請求。

**您的請求必須在考試計畫前2週發送。**

+++

+++如果失敗，我何時可以再次嘗試認證考試？

要獲得或保持您的Adobe認證狀態，您需要通過最新的認證考試，重新參加。 建議您在證書過期前留出足夠的時間考慮潛在的重新獲得。

對於首次嘗試Adobe認證考試不及格的應聘者，他們必須至少等待24小時再嘗試。 如果他們在第二次或後續嘗試中失敗，則在他們可以重新參加考試之前，需要等待14個日曆天。 每次嘗試都會產生單獨的考試費用。

+++

+++如何獲取分數報告副本？

* 步驟1:登錄到 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"},CertMetrics。 使用您的ID或電子郵件地址和密碼登錄。
* 步驟2:按一下「Trests（成績單）」。
* 第3步：滾動查看記錄歷史記錄並選擇要查看的報告。 您可以通過電子郵件將報告發送給您或在螢幕上查看報告。

+++

+++如何跟蹤我的認證狀態？

您可以通過轉到 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 的子菜單。

+++

+++通過認證考試後得到什麼？

完成考試後，結果將立即顯示在螢幕上。 在72小時內，您將從CertMetrics獲得對分數的電子郵件確認，並PDF您的證書。 您可以從CertMetrics帳戶訪問標誌，這些標誌可顯示您的認證狀態，並可用於您的網站、電子郵件、簽名、名片、社交檔案等。

+++

+++如何登錄或建立Adobe憑據管理/Certmetrics帳戶？

你可以 [登錄或在此處建立帳戶](https://www.certmetrics.com/adobe/){target="_blank"}。 您可以在此處查看可用的考試、您的考試歷史記錄、認證過期日期以及您可以享受的任何優惠（如折扣）。

+++

+++認證持續多長時間？

由於我們需要確保我們的Adobe認證人員在新的應用程式特性和功能上具有最新的Adobe認證 **有效期為兩年**。

+++

+++是否有英語以外的語言提供考試？

通過PSI（我們的供應商之一）提供一些日語考試。 如果日文有考試，將在該考試的考試詳細資訊中注明。 考試詳細資訊可在認證行程頁面的「準備就緒」頁籤上找到。

+++

+++您是否在Adobe活動中提供認證？

某些Adobe贊助的活動提供認證。

+++

+++認證是否與特定版本的應用程式(例如，Adobe Experience Manager6.5)相關聯？

不是直接的。 但是，由於認證現在需要每兩年續訂一次，因此您的認證可確保您保持最新的Adobe應用程式版本。

+++

>[!TAB 通過按需考試續訂]

+++什麼是認證續訂？ 如何續訂證書？

Adobe認證在兩年後到期。 您可以通過最新版的適用Adobe認證考試來續訂認證。

您有兩個選擇：

* 參加定期認證考試
* 參加續訂考試 — 一個更短的考試，重點介紹最新的解決方案功能以及您已經知道的內容。

如果您沒有原始證書或您的證書已過期，則不能參加續訂考試。

偶爾，我們會為某個角色而退休考試。 在這種情況下，您的認證將持續兩年，然後該認證將過期。 例如，我們不再提供Lead/Dev Ops認證，因此認證將在不進行替換測試的情況下過期。

+++

+++如果我在證書過期前不續訂，會發生什麼情況？

您的證書將過期，並在中顯示為過期 [!DNL Credly] 和你的社交資料。 此外，如果您的認證過期，則您不能參加較短的續訂考試或通過持續學習續訂。

瞭解有關如何管理徽章的詳細資訊 [!DNL Credly]按鈕 [這裡](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}。

+++

+++續訂我的證書需要多少成本？

您可以免費續訂您的證書。 這為您提供了一種經濟高效的方法來維護您的認證，並保持您的解決方案技能保持最新。

要瞭解有關如何續訂認證的更多資訊，請轉到 [這裡](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}。

+++

+++在哪裡可以查看我的證書過期日期？ 我能收到通知嗎？

登錄到 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"}。 在此，您可以看到您何時獲得憑據以及憑據何時過期。 您還應檢查您的個人資料，並確認您的聯繫人、輔助電子郵件和作業資訊是否正確。

您將在證書過期前一個月收到電子郵件通知。 但是，如果您已更換了工作，並且未更新電子郵件，則可能不會收到通知。 在您的CertMetrics帳戶中添加輔助電子郵件（如Gmail、Hotmail、Yahoo等），以便您獲得證書過期提醒電子郵件，即使您更改了公司。

+++

+++我需要參加哪項考試才能續課？

要瞭解您需要參加的考試，請登錄 [Adobe憑據管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 使用您的Adobe ID，按一下「認證」，然後選擇認證名稱。 續訂考試列於 **要求保持最新**。

+++

+++在續訂證書時，我是否會獲得新的數字徽章？

不可以。 您的數字徽章將保持不變。

+++

+++我能否參加與我參加初始認證相同的考試，以續簽認證？

不，在通過初始認證後，無法重新參加同一考試以續訂。

+++

>[!TAB 通過持續學習進行更新]

+++什麼是Adobe通過持續學習的認證更新計畫？

Adobe提供持續的學習和參與活動，以便候選人能夠保持活動，並將其認證地位延長2年。 這僅適用於選定的應用程式。

+++

+++我的證書已過期；我能否繼續開展持續學習和參與活動以進行更新？

不，此續訂選項僅適用於Adobe主動認證的分析、Experience Manager、商務和目標候選。

+++

+++哪些認證可提供持續學習和參與活動以進行續約？

目前，此計畫涉及分析、Experience Manager、商業和目標。 我們將通過適用渠道分享公告，並在我們擴展公告時提供其他認證。

+++

+++哪些持續學習和參與活動是認證續訂計畫的一部分？

將為候選人提供5至7門按需課程供審查。 續簽認證的標準將是通過基於課程的任何3項評估，或通過2項評估並提交經簽名的客戶參考表。

+++

+++在認證續訂中，什麼是客戶參考表？

作為Adobe持續學習和參與活動的一部分，候選人可提交的表格，以更新其證書。 該表格必須由客戶簽署，說明候選人在與他們正在更新其認證的產品相關的最近12個月內在項目上至少花費了100小時。 通過將此選項作為認證續訂要求的一部分，我們希望將您的持續參與視為驗證您技能和知識的一種方式，以便您的認證保持活動狀態。

+++

+++通過持續學習和參與活動進行認證續訂是否需要費用？

不可以。 所有活動都可通過Adobe憑據管理器免費訪問。

+++

+++我是否必須通過評估並提交客戶參考表？

否，提交客戶參考表是可選的，並且是候選人可以完成的續訂其證書的活動之一。 比如說， [!DNL Adobe Analytics] 證書續訂要求從給定清單中至少完成3個活動。 該清單包括通過6個評估並提交客戶參考表。 候選人可通過2次評估並提交客戶參考表，從而滿足續訂的3項最低要求。

+++

+++I在一個解決方案中擁有三個認證。 是否需要完成每個認證的要求？

不，您只需要完成該解決方案列出的要求。 完成所需活動並將結果提交到Adobe憑據管理器後，將續訂您的證書。 例如，如果您在 [!DNL Adobe Analytics] 作為業務從業人員、開發人員和建築師，您只需完成3項活動即可滿足所有要續訂的認證的續訂要求。

+++

+++我是需要複習課程，還是可以開始評估？

評估以Experience League課程為基礎，審查這些課程將使候選人能夠順利通過。 但是，續簽的關鍵要求只是通過評估。

+++

>[!TAB 重新啟動程式]

**計畫概述**

+++什麼是Adobe認證重新啟動程式？

Adobe認證重啟計畫是一次性計畫，允許個人通過線上短期考試或評估，免費重新獲得過期的Adobe認證。

+++

+++重新啟動程式將運行多長時間？

該方案將於2023年4月24日至10月20日持續六個月。

+++

+++Adobe以後會再次提供此類程式嗎？

目前，我們不打算再提這個。 這只是一次性產品。

+++

+++重新啟動計畫中包含哪些認證級別？

重新啟動計畫中只包含專家級和主級認證。 從2023年6月起，專業級認證將提供續訂途徑。

+++

**計畫資格**

+++誰有資格參加Adobe認證重啟計畫？

其認證從2020年到2023年4月23日過期的個人有資格參加重新啟動計畫，條件是他們尋求的認證已包括在該計畫中（請參閱下面的合格認證考試清單）。

+++

+++ 為什麼這些年來發佈的所有認證考試都未包括在「重新啟動」計畫中？\
2019年之前發佈的考試不包括在「重新啟動」程式中。

+++

+++為什麼2023年4月23日是重新啟動計畫資格的截止日期？

因為它允許無法參加免費認證續簽期（2023年1月23日啟動）的個人有足夠的時間續簽其認證。

+++

+++哪些認證是Adobe認證重啟計畫的一部分？

除已退休外，所有活動Adobe認證都是計畫的一部分。

+++我的認證將在2023年4月24日至10月20日之間過期，但此程式可用。一旦過期，我是否可以繼續重啟認證計畫？

不可以。 所有自推出之日起及以後具有有效認證的候選人都可以選擇繼續我們在2023年1月宣佈的持續學習續訂路徑。

重新啟動計畫僅適用於2023年4月24日之前過期的認證，以便向無法參加連續學習計畫的候選人提供認證。

**[!DNL Marketo Engage]解決方案架構師(MCSA)和Adobe Commerce**

+++I已過期 [!DNL Marketo] 認證解決方案架構師(MCSA)。 我可以通過此程式重新聲明嗎？

已過期的候選人 [!DNL Marketo] 認證解決方案架構師認證可以尋求Marketo業務從業者重新啟動路徑，以滿足對新解決方案的部分要求 [!DNL Marketo Engage] 建築師認證。

最後的要求是通過 [!DNL Marketo Engage] 建築師考試。 對於所有MCSA過期的證書持有者，我們在2023年12月31日之前將進行50%的折扣促銷，請與認證支援部門聯繫以瞭解詳細資訊。

+++

+++為什麼是Adobe [!DNL Commerce] 認證不是Adobe認證重啟計畫的一部分？

當前提供的Adobe沒有過期的證書持有者 [!DNL Commerce] 截至2023年4月23日的證明。 因此，所有活動Adobe [!DNL Commerce] 通過持續學習獲得認證的候選人有權選擇續訂 [認證的續訂路徑](https://experienceleague.corp.adobe.com/docs/certification/certification/technical-certifications/ac/ac-renew.html?lang=en){target="_blank"}。

**Adobe憑據管理器配置檔案**

+++我為什麼看到 _參與_ 在「Adobe憑據管理器」中的「我的配置檔案」的歷史記錄部分下？

這是為了標籤您是否有資格參加「Adobe認證重新啟動」計畫。 已分配此參與的候選人可以完成要求，在每個應用程式中重新啟動其過期的證書。

+++

+++I已完成要求，但在我的配置檔案歷史記錄中看不到結果，我何時才能獲得證書？

認證將在完成要求後72小時內獲得。 必須指出，本方案將只頒發合格候選人的認證。

+++

+++如果我的認證即使在完成重新啟動要求後仍顯示為過期，我該怎麼辦？

很遺憾，一旦證書過期，則無法擴展。 在這種情況下，發放新證書，而不是更新舊證書。
