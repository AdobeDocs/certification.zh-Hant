# Adobe認證常見問題集

若要深入瞭解Adobe認證計畫，請參閱以下的常見問題集(FAQ)。

![橫幅](/help/certifications/assets/home_banner_narrow.png)

>[!BEGINTABS]

>[!TAB 一般]

+++什麼是認證？有哪些優點？

個人優點：認證是辨識個人在Adobe應用程式中的技能與能力的認證。 通過認證考試的應徵者會成為Adobe認證專家、專家或碩士。

認證需要承諾、具備法律抗辯力，以及通過Adobe您的專業知識進行驗證。 您也可以從下載徽章，在社交設定檔中加入您的認證 [!DNL Credly].

若要進一步瞭解如何在中管理您的預算 [!DNL Credly]，執行 [此處](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++我如何知道該參加哪項考試？

認證是以角色為基礎，這表示有多個可用的考試。 檢閱認證歷程頁面，以選擇要參加的考試。 選擇考試的方式如下：

* 應用程式：例如， [!DNL Adobe Analytics] 或 [!DNL Marketo Engage]
* 角色：例如，開發人員或商務從業人員
* 等級：例如，專業，0-12個月的經驗；或專家1-3年

有時一個考試會有多個版本。 最好使用最新版本。

* 如果考試即將結束，您會在認證歷程頁面上看到備註。 這表示此測驗版本很快就會被移除，因為有新版本的測驗可供使用。 我們鼓勵應徵者使用最新版本，但如果您覺得更願意使用即將淘汰的版本，則可加以使用。 無論哪種情況，您的認證都將持續兩年。
* 如果您目前的認證即將到期，您應續約。 續約選項的問題較少，而且免費。 這些選項涵蓋最新的應用程式功能。 如果認證過期，您必須參加最新版本的認證考試，並自行付費。 [進一步瞭解更新選項](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++如果在監考期間遇到技術問題，該怎麼辦？

PSI和 [!DNL Examity] 提供聊天、電子郵件和電話支援。 Adobe支援無法協助解決考試期間發生的技術問題。

[!DNL Examity] 支援 - `adobepartnersupport@examity.com`

PSI支援 —  `psisupport@psionline.com`

最快速的說明資源是透過他們的聊天。 所有考生在參加考試前，應先確認電腦具備適當的系統需求。

如需詳細資訊，請參閱：PSI和 [!DNL Examity] 管理考試與技術設定的頁面。

+++

+++如有認證問題或問題，可以聯絡誰？

如有關於考試註冊、考試付款、考試傳送、測試中心、考試評論、考試問題、測試體驗、考試便利或有關認證入口網站的問題，請聯絡 `certif@adobe.com`.

+++

+++Adobe是否可容納有特殊需求的應徵者？

Adobe可滿足某些特殊需求。 如果您需要更多時間來完成考試或其他安排，請聯絡 `certif@adobe.com` 以您的要求。

**您的要求必須在考試排程前2週送出。**

+++

+++如果我考試不及格，何時可以再試一次？

若要取得或保留Adobe認證狀態，您必須重新參加最新的認證測驗。 建議您在憑證過期之前，預留足夠的時間來考慮可能的重新取得。

如果考生第一次通過Adobe認證考試時未通過，必須等待至少24小時，才能再次嘗試。 如果他們在第二次或任何後續嘗試中失敗，則需要14個日曆天的等待時間，才能重新參加考試。 每次嘗試都會產生個別的考試費用。

+++

+++如何取得分數報告的復本？

* 步驟1：登入 [Adobe認證管理系統](https://www.certmetrics.com/adobe/){target="_blank"}、CertMetrics。 使用您的ID或電子郵件地址和密碼登入。
* 步驟2：按一下「成績單」。
* 步驟3：捲動您的成績單歷史記錄，並選取您要檢視的報告。 您可以透過電子郵件將報告傳送給您，或是在畫面上檢視。

+++

+++如何追蹤我的認證狀態？

若要追蹤您的認證狀態，請前往 [Adobe認證管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 在「認證」標籤下。

+++

+++通過檢定測驗後會得到什麼？

完成測驗後，您的結果會立即顯示在熒幕上。 72小時內，您會收到一封電子郵件，確認您從CertMetrics取得的分數，並附上您憑證的PDF。 您可以從您的CertMetrics帳戶存取顯示您的認證狀態的圖志，並可用於您的網站、電子郵件、簽名、名片、社交設定檔等。

+++

+++如何登入或建立Adobe認證管理/Certmetrics帳戶？

您可以 [在這裡登入或建立帳戶](https://www.certmetrics.com/adobe/){target="_blank"}. 您可以在此檢視可用的考試、考試記錄、認證到期日以及任何優惠，例如折扣。

+++

+++認證持續多長時間？

由於我們需要確保經過Adobe認證的人員能及時瞭解新的應用程式功能，因此Adobe認證是 **有效期為兩年**.

+++

+++考試是否有英文以外的其他語言？

有些測驗可透過PSI （我們其中一家廠商）以日文提供。 如果測驗有日文版，則會在該測驗的詳細資訊中註明。 可在認證歷程頁面的「準備就緒」標籤上找到考試詳細資料。

+++

+++您是否在Adobe活動提供認證？

某些Adobe贊助的活動會提供認證。

+++

+++認證是否與特定版本的應用程式(例如Adobe Experience Manager 6.5)連結？

不直接，不。 但由於認證現在每兩年需要更新一次，因此您的認證可確保您保持最新的Adobe應用程式版本。

+++

>[!TAB 透過隨選考試續約]

+++什麼是認證續約？ 如何更新認證？

Adobe認證將在兩年後到期。 您可以參加適用的Adobe認證考試並取得最新版本以更新認證。

您有兩個選項：

* 參加定期的認證考試
* 參加更新考試 — 一種較短的考試，著重於最新的解決方案功能以及您已知道的事項。

如果您沒有原始憑證或憑證已過期，則無法參加續約考試。

我們偶爾會淘汰某個角色的考試。 在這種情況下，您的認證將持續兩年，然後該認證將過期。 例如，我們不再提供Lead/Dev Ops認證，因此認證將會過期，不需進行替代考試。

+++

+++如果我在憑證過期前沒有續約憑證會發生什麼事？

您的認證將會過期，並在以下位置顯示為已過期： [!DNL Credly] 以及您的任何社交設定檔。 此外，如果您的認證過期，就無法參加更短的續約測驗，也無法透過持續學習來續約。

若要進一步瞭解如何在中管理您的預算 [!DNL Credly]，執行 [此處](https://experienceleague.adobe.com/docs/certification/certification/how-to-get-certified.html?lang=en){target="_blank"}.

+++

+++更新認證的費用為何？

您可以免費續約您的認證。 這可讓您以符合成本效益的方式維護您的認證，並讓您的解決方案技能保持最新狀態。

若要進一步瞭解如何更新您的認證，請前往 [此處](https://experienceleague.adobe.com/docs/certification/certification/renew.html?lang=en){target="_blank"}.

+++

+++哪裡可以看見我的認證到期日？ 我會收到通知嗎？

登入 [Adobe認證管理系統](https://www.certmetrics.com/adobe/){target="_blank"}. 您可以在此處檢視取得認證的時間和到期時間。 您也應該檢查您的設定檔，並確認您的連絡人、次要電子郵件和工作資訊正確無誤。

您會在認證過期前一個月收到電子郵件通知。 但是，如果您變更了工作且沒有更新電子郵件，則您可能不會收到通知。 在您的CertMetrics帳戶中新增次要電子郵件（例如Gmail、Hotmail、Yahoo等），即使您變更公司，也可以收到認證到期提醒電子郵件。

+++

+++續約需要參加哪個考試？

若要瞭解您需要參加哪些考試，請登入 [Adobe認證管理系統](https://www.certmetrics.com/adobe/){target="_blank"} 使用您的Adobe ID，按一下「認證」，然後選取認證名稱。 更新測驗列於 **需要保持最新狀態**.

+++

+++續約認證時，我會得到新的數位徽章嗎？

不可以。 您的數位徽章將維持不變。

+++

+++我可以參加與初始認證相同的測驗來更新認證嗎？

不可以，一旦您通過初始認證後，就不能再重新參加相同的考試，以進行續約。

+++

>[!TAB 透過持續學習更新]

+++什麼是Adobe的認證續約計畫，透過持續學習？

Adobe提供持續學習和參與活動，讓應徵者可以保持活躍狀態並延長其認證狀態兩年。 這僅適用於選取的應用程式。

+++

+++我的認證已過期；我可以繼續學習和參與活動以進行續約嗎？

否，此續約選項僅適用於Adobe主動認證的Analytics、Experience Manager、Commerce和Target候選人。

+++

+++哪些認證可提供持續的學習與參與活動以進行續約？

目前此程式涉及Analytics、Experience Manager、Commerce和Target。 我們將透過適用管道分享公告，當我們將公告擴充至其他認證時。

+++

+++認證更新計畫包含哪些持續學習和參與活動？

候選人將獲得5至7個隨選課程以進行稽核。 續約認證的標準為通過任何3個以課程為基礎的評量，或通過2個評量，並提交已簽署的客戶推薦表。

+++

+++在認證更新的背景下，什麼是客戶參考表？

應徵者可以提交的表單，作為Adobe持續學習和參與活動的一部分，以更新其認證。 該表單必須由客戶簽署，宣告應徵者在他們要續約認證的產品相關的最近12個月內，已在專案上花費至少100小時。 我們將此選項納入認證更新要求中，藉此認可您持續參與的方式，來驗證您的技能和知識，讓您的認證保持有效。

+++

+++透過持續學習和參與活動追求認證更新是否需要費用？

不可以。 所有活動均可透過Adobe認證管理員免費存取。

+++

+++我是否必須通過評估並提交客戶推薦表？

否，提交客戶推薦表為選擇性，且應徵者可完成其中一項活動以更新其認證。 例如， [!DNL Adobe Analytics] 憑證續約需要完成指定清單中的至少3個活動。 此清單包括通過6項評估及提交客戶推薦表。 候選者可以通過2項評估並提交客戶推薦表，從而滿足至少3項續約要求。

+++

+++我在一個解決方案中擁有三項認證。 我是否需要完成每個認證的要求？

不需要，您只需要完成針對解決方案列出的需求。 完成所需活動並將結果提交給Adobe認證管理員後，您的認證將會續約。 例如，如果您在中持有多個認證 [!DNL Adobe Analytics] 身為業務從業人員、開發人員和架構師，您只需要完成3個活動即可符合所有認證的續約要求。

+++

+++我需要複習課程嗎？還是可以開始進行評量？

這些評量是以Experience League課程為基礎，稽核這些課程可讓考生順利通過評量。 不過，續約的主要要求就是要通過評鑑。

+++

>[!TAB 重新啟動程式]

**計畫總覽**

+++什麼是Adobe認證重新啟動計畫？

Adobe認證重新開始計畫是一次性計畫，可讓個人免費透過線上短期考試或評量重新取得過期的Adobe認證。

+++

+++重新啟動程式會執行多久？

此計畫將會在2023年4月24日至10月20日期間執行六個月。

+++

+++Adobe未來是否會再次提供此類計畫？

目前，我們不打算再次提供此功能。 此僅供一次性。

+++

+++重新啟動計畫包含哪些認證等級？

重新啟動計畫只包含專家級和主要級別的認證。 從2023年6月開始，專業級認證將推出續約路徑。

+++

**計畫資格**

+++誰符合Adobe認證重新啟動計畫的資格？

若個人的認證在2020年至2023年4月23日之間到期，只要他們尋求的認證包含在計畫中，便符合重新啟動計畫的資格（請參閱下方的「合格認證測驗」清單）。

+++

+++ 為什麼這些年發行的所有認證考試都不包含在重新開始計畫中？\
2019年之前發行的考試不包括在重新啟動計畫中。

+++

+++為什麼是2023年4月23日，是重新啟動計畫資格的截止日期？

因為此服務可讓無法參加免費認證續約期（2023年1月23日啟動）的個人有足夠的時間續約其認證。

+++

+++哪些認證是Adobe認證重新啟動計畫的一部分？

除了已淘汰的認證外，所有使用中的Adobe認證都是計畫的一部分。

+++我的認證將於2023年4月24日至10月20日到期，屆時此計畫將會推出。到期後，我可以繼續我的認證重新開始計畫嗎？

不可以。 所有在上市日期及之後取得有效認證的候選者，可選擇繼續沿用我們在2023年1月宣佈的持續學習更新路徑。

重新開始計畫僅適用於2023年4月24日之前過期的認證，以便為無法參加持續學習計畫的候選人提供認證。

**[!DNL Marketo Engage]解決方案設計師(MCSA)和Adobe Commerce**

+++我有一個過期的 [!DNL Marketo] 認證解決方案架構師(MCSA)。 我可以透過此程式重新陳述嗎？

具有已到期的候選者 [!DNL Marketo] 認證解決方案架構師認證可依循Marketo業務從業者重新開業的路徑，以符合新架構的部分需求 [!DNL Marketo Engage] 架構者認證。

最終要求是要通過新的 [!DNL Marketo Engage] 架構師考試。 對於所有MCSA過期憑證持有者，我們將在2023年12月31日之前執行50%折扣促銷，請連絡憑證支援以取得詳細資料。

+++

+++為什麼Adobe [!DNL Commerce] 認證不是Adobe認證重新啟動計畫的一部分嗎？

目前提供的Adobe沒有過期的憑證持有者 [!DNL Commerce] 2023年4月23日起的認證。 因此，所有作用中的Adobe [!DNL Commerce] 通過認證的應徵者可選擇透過持續學習續約 [憑證的續約路徑](https://experienceleague.corp.adobe.com/docs/certification/certification/technical-certifications/ac/ac-renew.html?lang=en){target="_blank"}.

**Adobe認證管理員設定檔**

+++為什麼我看到 _參與率_ 在Adobe認證管理員中我的設定檔的歷史記錄區段底下？

這是為了標示您符合Adobe認證重新啟動計畫的資格。 獲指派此參與資格的應徵者可在每個應用程式中完成重新開始其過期認證的要求。

+++

+++我已完成需求，但在我的設定檔歷程記錄中看不到結果，我何時會取得憑證？

完成需求後，將在72小時內授予此認證。 請務必注意，此計畫僅會授予合格候選人的認證。

+++

+++如果在完成重新啟動要求後，我的認證仍顯示為已過期，怎麼辦？

很遺憾，認證一旦過期，即無法延長。 在這種情況下，會核發新憑證，而不是更新舊憑證。
