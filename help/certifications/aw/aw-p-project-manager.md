# 認證Adobe [!DNL Workfront] 專業項目經理

了解如何成為認證的Adobe項目經理專業人員。

## 最低體驗

您應：

* 從初始工作請求到完成或接觸到項目或工作管理的對工作流程的理解
* 已完成自我引導培訓或訓練，或曾 [!DNL Workfront] 使用者6至12個月，讓他們了解基本功能
* 了解如何設定、追蹤及監控報表資料

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

>[!BEGINTABS]

>[!TAB 準備]

**目標受眾：**

* 項目經理、工作經理、計畫員
* SCRUM主
* [!DNL Workfront] 電源用戶
* 系統管理
* 計畫經理，產品組合經理
* Director，經理
* 資源管理員、流量管理員

**考試詳細資訊 — Adobe [!DNL Workfront] 專案經理**

* 級別：專業（0-12個月經驗）
* 傳遞分數：45/70
* 時間：140分鐘
* 傳送：線上監控（需要攝像頭訪問）
* 可用語言：英文
* 成本：125美元（全球）/95美元（印度）
* 考試ID:AD0-E903

**準備情況自評**

完成 [線上準備問卷](https://scorpion.caveon.com/launchpad/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional){target="_blank"} 來查看您目前的體驗是否符合建議的最低候選資格。

**考試目標和範圍**

第1節：入學和規劃(30%)

* 確定通知工作請求的不同方式
* 在提出請求時，請檢閱自訂資料以決定適當的動作
* 在案例中，決定活動所需的共用權限
* 在方案中，確定適當的任務設定，以規劃在特定時間範圍內的任務的執行
* 識別在專案上找出關鍵資料的方式
* 確定跟蹤項目風險的不同方法
* 在 [!DNL Workfront]
* 確定要查看特定分配資源的適當欄位
* 為角色指定，確定在項目/工作流程的工作接收或規劃階段中對角色有幫助的接收或規劃報告/控制面板

第二節：執行(50%)

* 識別了解、套用和測量專案里程碑的方式
* 建立 [!DNL Workfront] 單次使用核准和使用核准的時間
* 識別系統如何使用不同的日期類型
* 在方案中，確定日期類型向項目經理傳達的資訊
* 在方案中，確定使用不同任務約束時如何影響項目時間軸
* 在某種情況下，演示如何查找項目團隊以及如何通知他們工作分配、更新或完成工作項目
* 在案例中，說明如何上傳檔案和產生校樣
* 識別管理專案的不同方式
* 在方案中，確定單個項目中特定職務職責的計畫差異
* 在案例中，找出 [!DNL Workfront]
* 在甘特圖中，說明圖上的對象關係
* 比較條件和進度狀態之間的差異
* 識別方式 [!DNL Workfront] 團隊和 [!DNL Workfront] 可使用群組
* 在案例中，說明如何為利害關係人提供他們需要的資訊
* 在方案中，確定日期類型和持續時間對項目條件的影響
* 在案例中，決定需要自訂狀態
* 識別內可用的各種報表類型 [!DNL Workfront]
* 在案例中，說明建立自訂報表的步驟

第3節：結帳(20%)

* 在特定物件狀態下，決定需要執行哪些活動或觸發哪些動作
* 在方案中，確定用於修改任務清單或報告的報告類型和報告元素
* 識別可用於報表的不同萬用字元
* 識別可用於測量專案成功的專案元素
