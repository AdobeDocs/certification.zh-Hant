# 認證Adobe [!DNL Workfront] 項目經理專業人員

瞭解如何成為認證的Adobe項目經理專業人員。

## 最低體驗

您應該：

* 從最初的工作請求到完成或接觸到項目或工作管理，對工作流程的理解
* 已完成自導培訓或訓練營，或 [!DNL Workfront] 用戶6-12個月，以便他們瞭解基本功能
* 瞭解如何配置、跟蹤和監視報告資料

>[!NOTE]
>
>您的認證有效期為2年，從您通過認證之日起。 我們建議盡可能採用較新版本，因為它涵蓋較新的產品功能。

>[!BEGINTABS]

>[!TAB 準備好]

**目標受眾：**

* 項目經理、工作經理、計畫員
* SCRUM主
* [!DNL Workfront] 超級用戶
* 系統管理員
* 項目組經理，項目組經理
* Director，經理
* 資源管理器、流量管理器

**考試詳細資訊 — Adobe [!DNL Workfront] 項目經理**

* 級別：專業（0-12個月經驗）
* 通過分數：45/70
* 時間：140分鐘
* 交貨：線上監控（需要攝像頭訪問）
* 可用語言：英語
* 成本：125美元（全球）/95美元（印度）
* 考試ID:AD0-E903

**就緒性自我評估**

完成 [線上準備調查表](https://scorpion.caveon.com/launchpad/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional/ad-q-e903-readiness-questionnaire-for-adobe-workfront-project-manager-professional){target="_blank"} 查看您當前的經驗是否與推薦的最低候選人資格相符。

**考試目標和範圍**

第1節：吸收和規劃(30%)

* 確定通知工作請求的不同方式
* 如果有請求，請查看自定義資料以確定相應的操作
* 給定方案，確定活動所需的共用權限
* 給定方案，確定適當的任務設定以計畫在特定時間範圍內執行任務
* 確定在項目上查找關鍵資料的方法
* 確定跟蹤項目風險的不同方法
* 確定中對象的正確層次結構 [!DNL Workfront]
* 確定查看特定分配資源的相應欄位
* 給定角色，確定在項目/工作流程的工作接收或計畫階段對角色有幫助的接收或計畫報告/儀表板

第二節：執行(50%)

* 確定瞭解、應用和衡量項目裡程碑的方法
* 建立 [!DNL Workfront] 一次性使用批准和何時使用批准
* 確定系統如何使用不同的日期類型
* 給定方案，確定日期類型向項目經理傳達的資訊
* 給定方案時，確定使用時不同的任務約束對項目時間線的影響
* 給定一個方案，演示如何查找項目團隊以及如何將工作分配、更新或完成工作項目通知他們
* 給定方案，說明如何上載文檔並生成證明
* 確定管理項目的不同方法
* 給定方案，確定單個項目中特定任務職責的計畫差異
* 在給定方案時，在 [!DNL Workfront]
* 給定甘特圖，解釋圖表上的對象關係
* 比較條件和進度狀態之間的差異
* 確定如何 [!DNL Workfront] 團隊 [!DNL Workfront] 可以使用組
* 給定一個方案，說明如何向利益相關方提供所需資訊
* 給定方案，確定日期類型和持續時間如何影響項目條件
* 給定方案，確定需要的自定義狀態
* 確定中可用的各種類型的報告 [!DNL Workfront]
* 給定一個方案，描述構建自定義報告的步驟

第三節：結帳(20%)

* 給定特定對象狀態，確定需要執行哪些活動或觸發哪些操作
* 給定方案，確定用於修改任務清單或報表的報表類型和報表元素
* 確定報告中可使用的不同通配符
* 確定可用於衡量項目成功的項目元素

**參加認證考試**

Adobe使用兩家供應商來管理受監督的考試，即PSI和Examity。

* **PSI** 提供test中心和日語考試選項（如果可用）
* **[!DNL Examity]** 提供遠程線上選項

要計畫、重新安排或取消您的考試，只需按一下選擇的供應商，然後按照螢幕上提供的說明進行操作。 就這麼簡單！

Adobe憑據管理系統中的配置檔案名稱必須與政府ID完全匹配。 如果資訊不相同或不匹配，則檢查員將終止考試，並要求您除支付額外的考試費用外，還要重新安排約會。

如果在預定的考試約會前48小時之前開始，則可能取消和重新安排。 候選人必須自行修改此約會。
