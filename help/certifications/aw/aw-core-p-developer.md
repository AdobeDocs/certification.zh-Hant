# 認證Adobe [!DNL Workfront] 核心開發人員專業人員

瞭解如何成為Adobe Developer認證專家。

## 最低體驗

您應該：

* 至少3-6個月的使用Adobe [!DNL Workfront]
* 體驗Adobe [!DNL Workfront] 功能和連接性，包括接收、批准、規劃、執行和報告
* 對Adobe [!DNL Workfront] 對象類型，包括自定義表單和審批工作流、模板、持續時間、工作/計畫小時數
* 對Adobe [!DNL Workfront] 設定，包括訪問級別、佈局模板、項目首選項和其他方法，以簡化最終用戶設計
* 對流程映射的理解和體驗
* 對不同Adobe的理解 [!DNL Workfront] 角色和如何參與支援

而且，你應該：

* 熟悉 [!DNL Workfront] API和API瀏覽器，但不一定精通API調用
* 使用自定義API調用的能力 [!DNL Workfront] 和其他API
* 熟悉Adobe [!DNL Workfront]
* 熟悉本機連接器和其他Adobe [!DNL Workfront] 產品和服務
* 熟悉項目管理，包括靈活和瀑布以及資源管理概念

>[!NOTE]
>
>您的認證有效期為2年，從您通過認證之日起。 我們建議盡可能採用較新版本，因為它涵蓋較新的產品功能。

**目標受眾：**

* 技術顧問
* 系統管理員
* 項目經理
* 項目協調員
* 策略師
* 資源管理器
* 分析師
* IT專家
* [!DNL Workfront] 實施顧問/主管

**考試詳細資訊：**

* 級別：專業（0-12個月經驗）
* 通過分數：44/70
* 時間：140分鐘
* 交貨：線上監控（需要攝像頭訪問）
* 可用語言：英語
* 成本：125美元（全球）/95美元（印度）
* 考試ID:AD0-E905

**就緒性自我評估**

完成 [線上準備調查表](https://scorpion.caveon.com/launchpad/ad-q-e905-readiness-questionnaire-for-adobe-workfront-core-developer-professional/ad-q-e905-readiness-questionnaire-for-adobe-workfront-core-developer-professional){target="_blank"} 查看您當前的經驗是否與推薦的最低候選人資格相符。

**考試目標和範圍**

第1節 戰略和攝入量(24%)

* 使用項目組和程式
* 使用請求隊列
* 使用自定義表單
* 使用基本審批

第二節。 預算和規劃(11%)

* 使用資源計畫器
* 給定方案/需求集，應用配置以設定項目模板

第3節。 用戶體驗(25%)

* 說明在建立適當的用戶體驗時用戶管理的最佳做法
* 在給定情形下，應用整個系統共用和訪問權限的正確設定
* 描述Kickstart是什麼並識別何時使用它
* 演示如何建立和使用工時單
* 描述連接器、插件和載入項的用途和價值

第四節。 指標和報告(16%)

* 描述構建、發送和自定義報告的基本概念
* 解釋如何構建和自定義儀表板
* 演示如何配置報告的功能視圖

第五節。 執行(24%)

* 演示如何設定審批和校對工作流
* 確認為項目建立問題的目的
* 演示如何自定義組級項目狀態
* 確定啟用Agile所需的屬性 [!DNL Workfront]
* 確定在中輸入時間的方法 [!DNL Workfront]
* 確定中的財務跟蹤要素 [!DNL Workfront]

**參加認證考試**

Adobe使用兩家供應商來管理受監督的考試，即PSI和Examity。

* **PSI** 提供test中心和日語考試選項（如果可用）
* **[!DNL Examity]** 提供遠程線上選項

要計畫、重新安排或取消您的考試，只需按一下選擇的供應商，然後按照螢幕上提供的說明進行操作。 就這麼簡單！

Adobe憑據管理系統中的配置檔案名稱必須與政府ID完全匹配。 如果資訊不相同或不匹配，則檢查員將終止考試，並要求您除支付額外的考試費用外，還要重新安排約會。

如果在預定的考試約會前48小時之前開始，則可能取消和重新安排。 候選人必須自行修改此約會。
