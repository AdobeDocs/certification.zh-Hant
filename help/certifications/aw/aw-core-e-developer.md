# 認證Adobe [!DNL Workfront] 核心開發人員專家

了解如何成為認證的Adobe Developer專家。

## 最低體驗

您至少應有18至24個月的使用經驗 [!DNL Workfront]、和可：

* 導覽 [!DNL Workfront] 系統，了解其功能所在的位置
* 配置系統的所有方面
* 與客戶討論要求，並將這些要求轉化為可行的解決方案
* 以協作方式引發和修改現有流程以改進
* 了解並套用行銷和PMO使用案例
* 建議並建立支援客戶流程的報表

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

>[!BEGINTABS]

>[!TAB 準備]

**目標受眾：**

* [!DNL Workfront] 專門負責實作的系統管理員，以及新的或更新的執行個體設定
* 流程設計器/架構器
* 業務分析人員
* 技術顧問
* 配置顧問

**考試詳細資訊：**

* 級別：專家（1-3年經驗）
* 傳遞分數：36/64
* 時間：130分鐘
* 傳送：線上監控（需要攝像機訪問）或測試中心監控
* 可用語言：英文
* 成本：225美元（全球）/150美元（印度）
* 考試ID:AD0-E904

**準備情況自評**

完成 [線上準備問卷](https://scorpion.caveon.com/launchpad/ad-q-e904-readiness-questionnaire-for-adobe-workfront-core-developer-expert/ad-q-e904-readiness-questionnaire-for-adobe-workfront-core-developer-expert){target="_blank"} 來查看您目前的體驗是否符合建議的最低候選資格。

**考試目標和範圍**

第1節：核心繫統管理與設定(17%)

* 在案例中，說明有效共用權限和存取的方式。
* 給定圖，選擇正確的訪問層次順序。
* 在案例中，識別何時應使用群組狀態與系統狀態。
* 在資料集上，演示踢球開始的功能。
* 在案例中，找出自動布建使用者的最佳作法。
* 在案例中，說明使用者設定檔中的群組和團隊如何影響內可檢視或完成的項目 [!DNL Workfront].
* 在方案中，建議團隊應是標準、看板或掃描。
* 在案例中，決定部署Agile團隊自訂分析板的最佳實務。

第二節：取用、自訂表單和專案管理(17%)

* 在方案中，確定支援工作分配所需的配置區域。
* 在案例中，識別必要的設定。
* 在案例中，識別值如何傳遞至專案自訂表單。
* 在案例中，請說明必須執行哪些動作來解決報表和計算錯誤。
* 在案例中，找出適合範本的支援元素。

第3節：戰略功能 — 組合和方案管理，資源管理(13%)

* 在案例中，判斷如何擷取反映在Portfolio最佳化程式中的專案後KPI。
* 在某種情況下，調整業務案例以反映項目範圍的更改。
* 在案例中，建立一個檢視，用於識別專案是否符合要執行之必要條件的平均等級。
* 在一種情況下，描述如何定性或定量地管理風險。
* 在一種情況下，提供一種機制，確保最高優先項目的人員配備充足。
* 在案例中，確定根本原因。
* 在案例中，決定為何使用者分配過多。

第4節：文檔管理和校樣(13%)

* 在此情況下，演示何時使用文檔管理或校對
* 在案例中，找出使用校對檢視器、設定和標籤功能的最佳作法
* 在案例中，示範如何編輯和設定檔案的權限。
* 在某種情況下，演示如何設定具有安全限制的外部文檔儲存

第5節：報告(13%)

* 在案例中，說明將多欄資料合併成單一欄的方式。
* 在提供報表或篩選器時，說明使用萬用字元值的優點。
* 在報表中識別父子關係。
* 在某種情況下，描述報告如何促進工作的高效執行。
* 在案例中，說明 [!DNL Workfront] 報告以監控並改善使用者採用率。

第6節：方法/最佳作法/使用案例(20%)

* 在 [!DNL Workfront]，展示設定單一行銷活動中可交付項目追蹤功能。
* 在 [!DNL Workfront]，演示有關戰略優先順序和工作理由的功能。
* 在 [!DNL Workfront]，演示有關財務、利用、預測、計費率/記錄的功能
* 在 [!DNL Workfront]，展示設定工作流程和核准的相關功能。
* 在Agile的案例中，找出要顯示進度的原生功能。
* 在案例中，建議在展開用戶端例項時，以系統管理為核心進行控管的架構。
* 在某種情況下，在擴展客戶實例時確定需要考慮的區域，以使其他團隊能夠協同工作。

第7節：業務咨詢(8%)

* 在某種情況下，描述充分支援這項工作所需的過程。
* 在某種情況下，說明如何調整現有的工作流，以滿足關鍵的、即時的業務流程中最近的更改。
* 在案例中，識別最難變更為指定執行個體的組態設定。
* 在情形中，請確定 [!DNL Workfront] 實施
* 在情況下，請提供建議。

**參加練習測試**

建議您在開始之前參加免費練習測試。 您可以不受限制地訪問Adobe專家級考試的所有可用練習測試。

我們的練習測試都是從與現場考試相同的藍圖上開發的。 練習測試將幫助您了解即時考試中涵蓋的主題。

請記住，通過練習考試並不保證您將通過認證考試。

前往 [Adobe認證考試準備門戶](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 現在來做個練習測試。

**參加認證考試**

Adobe使用兩家供應商來管理監督考試（PSI和Examity）。

* **PSI** 提供測試中心和日文考試選項（如有）
* **[!DNL Examity]** 提供遠程聯機選項

要排程、重新計畫或取消您的考試，只需按一下所選供應商，然後按照螢幕上提供的說明操作即可。 就這麼簡單！

### 考試次數：AD0-E904
