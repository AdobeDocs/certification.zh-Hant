# 認證Adobe [!DNL Campaign Standard] 開發人員專家

瞭解如何成為認證的Adobe Developer專家。

## 最低體驗

您應該：

* 至少兩年 [!DNL Campaign] 軟體體驗
* 在多渠道數字營銷活動和Web開發（如REST API）方面至少擁有兩年的相關經驗
* 能夠在Adobe Campaign執行包括模板開發、技術工作流、模式開發和整合在內的任務，並能夠使用控制面板監視實例
* 在渠道交付環境（電子郵件、直郵、移動和社交）中工作的體驗
* 熟悉多點觸控、生命週期和重新激活等營銷活動策略
* 瞭解市場營銷指標，如ROI、點擊率和轉換。
* 管理營銷自動化平台的經驗
* CSS、HTML和JavaScript的基本知識，電子郵件
* 關係資料庫基礎知識與資料建模
* 瞭解技術工作流並配置它們
* 擴展收件人表或方案和資料管理的經驗
* 已建立類型規則、工作流、交付和市場活動模板
* 配置的角色和權限
* 在管理控制台和與其他Adobe應用程式整合方面的經驗
* 對Adobe I/O和Adobe I/O Runtime的認識
* 在多渠道上處理事務性消息傳遞和消息傳遞方面的經驗
* 受管理的包的體驗

>[!NOTE]
>
>您的認證有效期為2年，從您通過認證之日起。 我們建議盡可能採用較新版本，因為它涵蓋較新的產品功能。

**考試詳細資訊：**

* 級別：專家（1-3年經驗）
* 通過分數：29/50
* 時間：100分鐘
* 交貨：線上監控（需要攝像頭訪問）或test中心監控
* 可用語言：英語
* 成本：225美元（全球）/150美元（印度）
* 考試ID:AD0-E306

**就緒性自我評估**

**考試目標和範圍**

第1節：資料建模和管理(26%)

* 給定一個方案，將客戶端需求轉換為自定義資源
* 確定如何配置螢幕和篩選器定義
* 給定方案，確定定義鍵、索引和連結的正確策略
* 確定發佈資源更新的正確方法
* 排除資料建模問題
* 確定構建技術工作流的正確方法

第二節：市場活動管理(50%)

* 排除工作流錯誤
* 確定構建市場活動的正確方法
* 確定與受眾和資料協作的正確方法
* 通過類型應用和建立篩選器規則
* 確定建立和管理模板的正確方法
* 確定配置登錄頁的正確方法
* 確定正確的服務和訂閱方法
* 假設情況，驗證交付機制
* 確定使用創造性設計器構建電子郵件的正確方法
* 確定建立自定義報告的正確方法

第三節：行政(24%)

* 確定建立用戶安全組和組織單位的正確方法
* 確定管理和部署軟體包的正確方法
* 確定配置外部帳戶的正確方法
* 確定配置選項的正確方法
* 確定建立品牌的正確方法
* 確定開發渠道配置的正確方法

**現在您已成功登錄，將直接訪問以下連結。**

**練習test**

我們建議在您開始之前先享受免費練習test。 您將可以不受限制地訪問所有可用的練習test，以便進行Adobe的專家級考試。

我們的實踐test都是從與現場考試相同的藍圖發展出來的。 練習test將幫助您瞭解即時考試所涵蓋的主題。

請記住，通過練習test並不保證您會通過認證考試。

轉到 [Adobe認證考試準備門戶](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 現在去做一個練習test。

**參加認證考試**

Adobe使用兩家供應商來管理受監督的考試，即PSI和Examity。

* PSI提供test中心和日文考試選項（如有）
* 示例提供遠程線上選項

要計畫、重新安排或取消您的考試，只需按一下選擇的供應商，然後按照螢幕上提供的說明進行操作。 就這麼簡單！

Adobe憑據管理系統中的配置檔案名稱必須與政府ID完全匹配。 如果資訊不相同或不匹配，則檢查員將終止考試，並要求您除支付額外的考試費用外，還要重新安排約會。

如果在預定的考試約會前48小時之前開始，則可能取消和重新安排。 候選人必須自行修改此約會。
