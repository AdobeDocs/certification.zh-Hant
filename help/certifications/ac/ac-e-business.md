# 認證Adobe [!DNL Commerce] 業務從業專家

了解如何成為認證的Adobe業務從業專家。

## 最低體驗

你應該有約1-3年的實踐 [!DNL Commerce] 對電子商務體驗有廣泛了解。

您應該有下列方面的經驗：

* 數位行銷：SEO, CMS，交叉銷售和追加銷售等銷售技術，了解Google Analytics，電子郵件模板
* 安全性、法規遵循和隱私權法，例如PCI、GDPR、ADA、CAN-SPAM、PSD2
* 全通路和多頻道等銷售/行銷策略
* 項目管理、需求收集、整合流程、一般電子商務法、軟體開發方法和監控方面的經驗
* 了解商業、商業/電子商務和物流
* 對可定製應用程式、雲和伺服器基礎架構的基本了解
* 現代電子商務術語

此外，您也應熟悉使用下列技術：

* Adobe [!DNL Commerce]  — 版本2.4
* 分析和效能工具：Google Analytics、標籤管理程式、SWAT、New Relic、Ampley、Google燈塔
* 安全工具：PCI法規遵循、ADA、GDPR等
* 雲端或內部部署架構

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

**目標受眾：**

* 業務分析人員
* 商務策略師
* 顧問
* 技術專案經理/技術客戶經理
* 應用程式架構師/解決方案架構師/專家
* 代理管理員/Director/經理/顧問
* Digital Marketing Director/經理/開發人員

**考試詳細資訊：**

* 級別：專家（1-3年經驗）
* 傳遞分數：31/50
* 時間：100分鐘
* 傳送：線上監控（需要攝像機訪問）或測試中心監控
* 可用語言：英文
* 成本：225美元（全球）/150美元（印度）
* 考試ID:AD0-E708

**準備情況自評**

完成 [線上準備問卷](https://scorpion.caveon.com/launchpad/ad-q-e708-readiness-questionnaire-for-adobe-commerce-business-practitioner-expert-exam/ad-q-e708-readiness-questionnaire-for-adobe-commerce-business-practitioner-expert-exam){target="_blank"} 來查看您目前的體驗是否符合建議的最低候選資格。

**考試目標和範圍**

第1節：核心功能/一般配置(46%)

* 識別Adobe功能 [!DNL Commerce] 開放原始碼版本和 [!DNL Commerce] 版本
* 區分所有版本的Adobe [!DNL Commerce] 產品
* 確定如何利用產品類型及其功能來滿足客戶需求
* 解譯需求和模型，以判斷是否符合原生功能
* 展示「管理面板」的相關知識和常見功能的位置
* 演示導入/導出Adobe [!DNL Commerce] 實體
* 了解如何以原生方式設定購物車和結帳
* 在Adobe中評估本機可用的發運方法 [!DNL Commerce] 及其如何適用於一般使用案例
* 了解使用Adobe建立和發佈樣式化內容的方式 [!DNL Commerce] CMS功能，包括PageBuilder
* 使用原生工具管理訂單生命週期
* 展示配置各種送禮選項（禮品卡、禮品包裝、贈送訊息）的能力
* 設定和修改交易式電子郵件
* 說明B2B中的客戶自助服務和忠誠計畫原生功能

第二節：銷售(10%)

* 展示建立促銷活動以符合特定業務標準的能力，以及如何決定最終定價
* 展示管理類別和產品的能力
* 了解不同的定價設定，以及它們如何影響最終價格

第3節：數位行銷(4%)

* 建議使用原生功能的SEO最佳作法
* 評估Google Analytics和BI中的通用量度

第4節：附加模組和附加產品(18%)

* 說明B2B功能，及其與常見B2B案例的關聯方式
* 應用業務要求以建議使用MSI的解決方案
* 說明BI的優點以及如何使用BI來Adobe [!DNL Commerce] 解決方案
* 區分原生搜尋和LiveSearch的差異
* 應用業務要求以確定如何在B2B環境中應用稅、關稅和免稅
* 了解如何為B2B客戶套用量身打造的定價
* 了解Adobe [!DNL Commerce] 原生產品和Adobe Sensei產品建議

第5節：系統體系結構(10%)

* 評估需求，以確定哪些網站、商店和商店檢視是必要的
* 識別並分析效能指標以進行改進
* 了解可將外部系統與Adobe整合的可用方法 [!DNL Commerce]
* 區分無頭方法與傳統方法

第6節：合規性/安全性(12%)

* 演示如何保護Adobe [!DNL Commerce] 具有角色和權限的資料存取
* 了解隱私權法與支付安全性的法規遵循基礎
* 說明Adobe的常見安全性 [!DNL Commerce] 專案
* 了解稅法的基本概念及如何設定

**現在您已成功登入，您將可直接存取下列連結。**

**參加練習測試**

建議您在開始之前參加免費練習測試。 您可以不受限制地訪問Adobe專家級考試的所有可用練習測試。

我們的練習測試都是從與現場考試相同的藍圖上開發的。 練習測試將幫助您了解即時考試中涵蓋的主題。

請記住，通過練習考試並不保證您將通過認證考試。

前往 [Adobe認證考試準備門戶](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 現在來做個練習測試。

**參加認證考試**

Adobe使用兩家供應商來管理監督考試（PSI和Examity）。

* **PSI** 提供測試中心和日文考試選項（如有）
* **[!DNL Examity]** 提供遠程聯機選項

要排程、重新計畫或取消您的考試，只需按一下所選供應商，然後按照螢幕上提供的說明操作即可。 就這麼簡單！
