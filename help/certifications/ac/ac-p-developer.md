# 認證Adobe [!DNL Commerce] 開發人員專業人員

學習如何獲得Adobe Developer專業認證。

## 最低體驗

你應該很自在地工作Adobe [!DNL Commerce] 對Adobe有基本認識 [!DNL Commerce] 管理開發、體系結構、EAV/資料庫、佈局/UI、結帳和銷售，以及目錄。

此外，您應熟悉以下技術和環境：

* Adobe [!DNL Commerce]  — 版本2.4
* 對PHP的理解
* MYSQL知識
* 物件導向程式設計知識
* 雲架構知識

>[!NOTE]
>
>您的認證有效期為2年，從您通過認證之日起。 我們建議盡可能採用較新版本，因為它涵蓋較新的產品功能。

>[!TAB 準備好]

**目標受眾：**

* 開發人員
* 領先開發人員
* 後端開發人員
* 解決方案架構師
* 技術領袖

**考試詳細資訊：**

* 級別：專業（0-12個月經驗）
* 通過分數：50/77
* 時間：154分鐘
* 交貨：線上監控（需要攝像頭訪問）
* 可用語言：英語
* 成本：125美元（全球）/95美元（印度）
* 考試ID:AD0-E717

**就緒性自我評估**

完成 [線上準備調查表](https://scorpion.caveon.com/launchpad/ad-q-e717-readiness-questionnaire-for-adobe-commerce-developer-professional/ad-q-e717-readiness-questionnaire-for-adobe-commerce-developer-professional) 查看您當前的經驗是否與推薦的最低候選人資格相符。

**考試目標和範圍**

第1節：使用管理員(5%)

* 描述ACL如何與角色和資源配合工作
* 標識建立或修改管理網格/表單時要使用的元件
* 標識建立儲存/管理配置和菜單項時要使用的檔案

第二節：體系結構(29%)

* 描述 [!DNL Magento] 檔案結構
* 描述 [!DNL Magento] CLI命令
* 描述cron功能
* 給定方案，描述di.xml的用法
* 在給定方案時，建立控制器
* 描述模組結構
* 描述索引功能
* 描述本地化
* 描述插件、首選項、事件觀察器和截取器
* 描述自定義模組路由
* 描述URL重寫
* 描述 [!DNL Magento] 快取系統
* 描述商店、網站和商店視圖（基本理解）

第三節：EAV/資料庫(13%)

* 給定方案，更改/添加/刪除屬性集和/或屬性
* 描述不同類型的屬性
* 給定方案時，使用DB架構更改資料庫表
* 描述模型、資源模型和集合
* 描述實體屬性值(EAV)的基礎知識

第四節：佈局/UI(15%)

* 將更改應用於現有產品類型並建立新產品類型
* 修改和擴展現有目錄實體
* 演示管理索引和自定義價格輸出的能力
* 解釋多來源庫存如何影響庫存（方案水準）

第五節：結帳和銷售(8%)

* 描述購物車元件
* 描述購物車促銷規則
* 給定方案，描述基本的簽出修改
* 給定一個方案，描述報價資料的基本用法
* 在給定方案下，配置付款和發運方法
* 給定方案，配置稅則、幣種、購物車和/或結帳

第6節：目錄(8%)

* 確定類別管理和產品管理的基本知識
* 描述產品類型
* 描述價格規則
* 描述價格類型

第七節：Adobe [!DNL Commerce] 雲體系結構(12%)

* 定義Adobe [!DNL Commerce] 體系結構/環境工作流
* 描述雲項目檔案、權限和結構
* 列出可用的Adobe [!DNL Commerce] 在雲上
* 描述如何訪問不同類型的日誌
* 描述應用修補程式的步驟（確定要放入修補程式的資料夾）
* 介紹如何維護和升級ECE工具
* 確定何時呼叫支援*Yaml檔案和限制（DIY與支援票證）

第八節：設定/配置Adobe [!DNL Commerce] 雲(4%)

* 確定如何設定/配置Adobe [!DNL Commerce] 雲
* 定義基本雲疑難解答（Web UI和變數的層次結構、配置優先順序）
* 識別雲用戶管理和登錄UI的基本知識
* 使用UI描述環境管理
* 使用UI描述分支
* 確定Adobe商務雲計畫功能

第九節： [!DNL Commerce] 雲CLI工具（管理部件）(6%)

* 描述獨有的Adobe [!DNL Commerce] 雲CLI工具(CLI獨有功能：激活電子郵件、重新建立環境、快照、資料庫轉儲、本地環境設定)
* 使用Adobe描述分支 [!DNL Commerce] 雲CLI工具
* 確定連接到雲服務的方法？ （我的SQL、Redis、tunnel:info）

**參加認證考試**

Adobe使用兩家供應商來管理受監督的考試，即PSI和Examity。

* **PSI** 提供test中心和日語考試選項（如果可用）
* **[!DNL Examity]** 提供遠程線上選項

要計畫、重新安排或取消您的考試，只需按一下選擇的供應商，然後按照螢幕上提供的說明進行操作。 就這麼簡單！
