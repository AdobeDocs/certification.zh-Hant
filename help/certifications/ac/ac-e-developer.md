# 認證Adobe [!DNL Commerce] 開發人員專家

了解如何成為認證的Adobe Developer專家。

## 最低體驗

您至少應有1至3年的Adobe經驗 [!DNL Commerce] 開發，您熟悉下列技術和環境：

* Adobe [!DNL Commerce]  — 版本2.4
* PHP
* XDebug/PHP調試工具，瀏覽器調試工具
* 連結工具
* 撰寫器
* SSH，基本Linux
* 雷迪斯，Elasticsearch, SQL，清漆，RabbitMQ
* API驗證工具， API用戶端工具(Postman)
* 版本控制軟體，例如Git
* 雲架構

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

>[!BEGINTABS]

>[!TAB 準備]

**目標受眾：**

* 開發人員
* 潛在開發人員
* 後端開發人員
* 解決方案架構師
* 技術領導者

**考試詳細資訊：**

```
Level: Expert (1-3 years' experience)
Passing Score: 41/69
Time: 138 mins
Delivery: Online proctored (requires camera access)
Available languages: English
Cost: $225 (global) / $150 (India)
Exam ID: AD0-E716
```

**準備情況自評**

完成 [線上準備問卷](https://scorpion.caveon.com/launchpad/ad-q-e716-readiness-questionnaire-for-adobe-commerce-developer-expert/ad-q-e716-readiness-questionnaire-for-adobe-commerce-developer-expert){target="_blank"} 來查看您目前的體驗是否符合建議的最低候選資格。

**考試目標和範圍**

第1節：Adobe [!DNL Commerce] 架構與自訂技術(36%)

* 演示如何在Adobe中有效使用快取 [!DNL Commerce]
* 建立、使用和操控自訂擴充功能屬性
* 建議如何將主題自訂更新套用至產品/類別頁面的解決方案
* 正確套用觀察者、偏好設定和外掛程式（排序順序的效果）
* 演示在Adobe中使用配置層的能力 [!DNL Commerce]
* 展示路由如何在Adobe中運作的知識 [!DNL Commerce]
* 展示自訂頁面建立工具的功能
* 決定在單一執行個體上設定多個網站的效果和限制
* 描述依賴插入的功能和約束
* 說明如何新增和設定存放區設定中的欄位
* 說明Git修補程式的使用案例以及撰寫器中的檔案層級修改
* 在CLI中建立新命令
* 演示如何編寫整合測試
* 識別Adobe [!DNL Commerce] 安全性功能（CSP、逸出、表單金鑰、衛生處理、reCAPTCHA、輸入驗證）
* 說明CRON排程系統如何運作
* 展示載入和操控資料的能力
* 展示使用應用程式模擬的能力

第二節：使用資料庫和EAV(10%)

* 以寫程式方式操控EAV屬性和屬性集
* 演示擴展資料庫架構的能力
* 展示從Adobe Commerce匯入/匯出資料的能力
* 說明如何使用修補程式和循環設定來修改資料庫

第3節：與管理員一起開發(6%)

* 演示更新和建立網格和表單的能力
* 擴展網格操作
* 演示建立修飾詞類的能力
* 演示限制訪問ACL的能力

第4節：自訂目錄(6%)

* 將變更套用至現有的產品類型並建立新的產品類型
* 修改和擴展現有目錄實體
* 演示管理索引和自定義價格輸出的能力
* 說明多來源庫存如何影響庫存（方案層）

第5節：定制銷售操作(6%)

* 展示開發新支付方法或定制現有支付方法的能力
* 演示添加和自定義發運方法的能力
* 演示定制銷售操作的能力
* 說明如何自訂總計

第6節：API與服務(4%)

* 展示建立新API或擴充現有API的能力
* 演示使用排隊系統的能力

第7節：Adobe [!DNL Commerce] 雲架構(16%)

* 展示Adobe知識 [!DNL Commerce] 架構/環境工作流程
* 展示雲端專案檔案、權限和結構的實用知識
* 演示根據Adobe設定多域儲存的能力 [!DNL Commerce] 雲（多網域網站結合開發工作與支援）
* 展示應用程式服務的一般知識，以及如何管理應用程式服務(YAML 、 PHP 、 MariaDB 、 Redis 、 RabbitMQ等)
* 識別如何存取不同類型的記錄檔
* 演示部署項目的能力（主要部署步驟）
* 定義ECE工具提供的功能
* 識別ECE修補程式的用途（安全違規）
* 說明如何維護和升級ECE工具
* 區分何時聯繫支援*Yaml檔案和限制（DIY與支援票證）
* 演示OOTB ABMITLY功能配置和安裝的基本知識

第8節：設定/設定Adobe [!DNL Commerce] 雲(10%)

* 說明如何設定/設定Adobe [!DNL Commerce] 雲
* 套用基本雲端疑難排解知識（Web UI和變數的階層，設定優先）
* 展示對雲端使用者管理和入門UI的了解
* 說明如何使用UI更新雲端變數
* 使用UI說明環境管理
* 使用UI展現對分支的了解
* 識別Adobe商務雲計畫功能

第9節： [!DNL Commerce] Cloud CLI工具（管理部件）(6%)

* 演示使用CLI更新雲變數的了解
* 演示使用CLI了解環境管理（CLI獨有功能：激活電子郵件、重新建立環境庫、快照、資料庫轉儲、本地環境設定）

**參加練習測試**

建議您在開始之前參加免費練習測試。 您可以不受限制地訪問Adobe專家級考試的所有可用練習測試。

我們的練習測試都是從與現場考試相同的藍圖上開發的。 練習測試將幫助您了解即時考試中涵蓋的主題。

請記住，通過練習考試並不保證您將通過認證考試。

前往 [Adobe認證考試準備門戶](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 現在來做個練習測試。

**參加認證考試**

Adobe使用兩家供應商來管理監督考試（PSI和Examity）。

* **PSI** 提供測試中心和日文考試選項（如有）
* **[!DNL Examity]** 提供遠程聯機選項

要排程、重新計畫或取消您的考試，只需按一下所選供應商，然後按照螢幕上提供的說明操作即可。 就這麼簡單！

### 考試次數：AD0-E716
