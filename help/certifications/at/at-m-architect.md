# 的認證歷程 [!DNL Adobe Target] 架構師主

了解如何成為認證的Adobe架構師主師。

## 最低體驗

您應：

* 至少三年的數位行銷最佳化體驗，包括實作 [!DNL Adobe Target] 活動
* 實作、部署和疑難排解的經驗至少為兩年（或5次參與） [!DNL Adobe Target]
* 體驗整合Target與Adobe Experience Cloud
* 對JavaScript、HTML、CSS和前端Web開發的基本了解
* 熟悉/了解 [!DNL Adobe Target] API/SDK、伺服器端整合、報告及管理
* 實作知識 [!DNL Adobe Target] 搭配單頁應用程式(SPA)架構、行動裝置和電子郵件，以及建議

此外，您也應熟悉下列技術和環境：

技術：

* 多個Adobe Experience Cloud程式(Adobe Analytics、Adobe Campaign、Adobe Experience Manager、Adobe Experience Cloud Debugger、Target API)
* JavaScript（Ajax、JQuery等）
* 以瀏覽器為基礎的除錯工具（Chrome開發人員工具、VEC helper）、JSON，用於偵錯來自Target的回應
* 原生應用程式的除錯工具(Charles Proxy, Fiddler)
* HTML, CSS, HTTP, XML配置
* 原生行動開發IDE、iOS開發基本概念、Android開發基本概念
* API工具，例如Postman
* VEC helper

環境:

* Adobe Experience Cloud
* 不使用標籤管理程式的用戶端實作
* Tag Management系統，尤其是Launch(IF)
* 伺服器端環境（供API使用）
* 電子郵件
* 網頁和行動環境
* AMS托管環境
* Adobe I/O Runtime

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

**目標受眾：**

* 轉換率最佳化工程師/架構師
* 個人化工程師/架構師
* 數位解決方案架構師
* 數位轉換顧問
* 體驗最佳化銷售機會
* 最佳化管理員
* 線上行銷活動管理員

**考試詳細資訊：**

* 級別：碩士（3-5年經驗）
* 傳遞分數：34/50
* 時間：100分鐘
* 傳送：線上監控（需要攝像機訪問）或測試中心監控
* 可用語言：英文
* 成本：225美元（全球）/150美元（印度）
* 考試ID:AD0-E407

**考試目標和範圍**

第1節： [!DNL Adobe Target] 基本面(12%)

* 識別適當 [!DNL Adobe Target] 特定情境的使用情況
* 說明 [!DNL Adobe Target] 實施

第二節：與整合 [!DNL Adobe Target] (16%)

* 應用流程以整合 [!DNL Adobe Target] 與Adobe Analytics(A4T)
* 應用流程以整合 [!DNL Adobe Target] 到Adobe Experience Manager
* 說明與其他Adobe Experience Cloud解決方案的整合
* 說明與Adobe Experience Cloud Audiences的整合

第3節： [!DNL Adobe Target] 行政和治理(13%)

* 定義持續的 [!DNL Adobe Target] 執行戰略和系統治理
* 套用程式和最佳實務以管理使用者
* 說明實作設定的管理方面

第4節： [!DNL Adobe Target] 和支援的接觸點(15%)

* 說明用戶端傳送實作
* 說明伺服器端傳送實作
* 說明行動裝置、電子郵件鎖定目標和API的實作需求

第5節：運用 [!DNL Adobe Target] 建議(9%)

* 展示對建議、目錄和環境的了解
* 套用程式以設定Target建議

第6節：架構、設計和實作 [!DNL Adobe Target] (15%)

* 選取正確的mbox實作
* 演示設計 [!DNL Adobe Target] 實施
* 說明的實作 [!DNL Adobe Target] 原生行動應用程式中

第7節：疑難排解(20%)

* 疑難排解mbox錯誤
* 疑難排解活動錯誤
* 了解整合錯誤
* 疑難排解 [!DNL Adobe Target] Recommendations錯誤
