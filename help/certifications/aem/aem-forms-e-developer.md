# 認證Adobe [!DNL Experience Manager Forms] 開發人員專家

了解如何取得認證 [!DNL Experience Manager Forms] 開發人員專家。

## 最低體驗

您應：

* 1-3年AEM Forms經驗
* IT行業至少5年的工作經驗
* 能夠領導定制表單解決方案的開發、配置、測試和部署
* 能夠配置表單、將業務需求轉化為工作流、與外部系統整合
* 了解AEM Forms基礎元件

此外，您也應熟悉下列技術和環境：

* 設計XDP範本、XFA範本
* 建立PDF forms（靜態/動態、互動式和非互動式）
* 架構或XML資料綁定
* 建立表單片段
* 建立最適化表單
* 複雜表單的調試
* 設定並執行表單提交
* 針對最適化表單建立和部署AEM例項
* 使用表單模型
* 與資料服務整合的能力
* 工作流程實作
* 基本了解AEM forms workbench(JEE)，以建立長存留期 — 短暫的處理程式和服務。
* 使用GuideBridge API開發Javascript用戶端
* 根據簽署v政府CAC ，知道要使用哪個AEM Forms環境案頭或伺服器
* 基本了解Java/J2EE和JavaScript，以編寫邏輯並呼叫不同的服務。
* 預填最適化表單
* OSGi、Sling元件的基本了解或知識
* 文檔服務的使用
* AEM Designer, AEM Workbench, AEM工作區
* 適用性表單主控台
* Java IDE工具
* 馬文
* GIT/版本控制工具

>[!NOTE]
>
>你的認證有效期為兩年。 建議您盡可能使用較新版本，因為它涵蓋較新的產品功能。

**考試目標和範圍**

第1節：Forms建立與管理(36%)

* 建置Forms
* 設定Forms資料模型
* 自訂AEM Forms
* 運用GuideBridge API開發及使用Javascript ClientLib

第二節：程式和工作流程(34%)

* 設計並協調表單核准工作流程
* 使用Document Services API開發OSGi套件組合
* 使用Watchfolder端點排程不同的非同步批次程式
* 從XFA/XDP設計中產生不同的輸出
* 使用Forms或組合器服務操控PDF
* 實施JDBC服務

第3節：管理和疑難排解(18%)

* 疑難排解和區別記錄檔
* 執行管理活動
* 應用Service Pack
* 為Forms執行AEM Dispatcher設定
* 導覽不同的主控台

第4節：API端點和整合(8%)

* 整合 Adobe Experience Cloud
* 生成和使用各種端點服務

第5節：互動式通訊(4%)

* 設計、展示和實作互動式通訊的程式

**現在您已成功登入，您將可直接存取下列連結。**

**參加練習測試**

建議您在開始之前參加免費練習測試。 您可以不受限制地訪問Adobe專家級考試的所有可用練習測試。

我們的練習測試都是從與現場考試相同的藍圖上開發的。 練習測試將幫助您了解即時考試中涵蓋的主題。

請記住，通過練習考試並不保證您將通過認證考試。

前往 [Adobe認證考試準備門戶](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} 現在來做個練習測試。

**參加認證考試**

Adobe使用兩家供應商來管理監督考試（PSI和Examity）。

* **PSI** 提供測試中心和日文考試選項（如有）
* **[!DNL Examity]** 提供遠程聯機選項

要排程、重新計畫或取消您的考試，只需按一下所選供應商，然後按照螢幕上提供的說明操作即可。 就這麼簡單！
