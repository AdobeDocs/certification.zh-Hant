# 認證Adobe [!DNL Experience Manager Sites] 建築師大師

瞭解如何成為認證建築師主管。

## 最低體驗

您應該：

* 至少3年建築AEM Sites的建築師經驗
* 在Adobe Managed Services和Adobe中工作的能力 [!DNL Experience Manager as a Cloud Service]
* 設定Adobe [!DNL Experience Manager] 環境和基礎架構

此外，您應具備執行以下任務的能力：

* 為客戶設計軟體更改流程（在前和雲）
* 為內容/資產定義一個可適應未來需求的內容模型
* 根據業務需求定義必要的模板和元件
* 確定何時使用開箱即用/功能/核心元件而不是自定義元件
* 設計快取、負載平衡和安全策略（分發程式、防火牆、CDN）
* 為初始設定和功能交付提供高級工作量估計
* 建立國際化的功能與使用案例
* 定義支援多站點和多語言的策略
* 確定軟體模組的正確配置機制
* 將第三方和Adobe產品與
* 定義遷移和升級策略（產品和內容）
* 瞭解部署方法（內容和代碼）
* 建立和提供基礎架構/部署圖
* 確定要與應用程式一起使用的信AEM息模型
* 定義身份驗證和授權安全模型和概念
* 根據客戶的要求（二進位儲存、部署方案等）確定持久性模型
* 為業務流程設計工作流模型
* 制定平台效能和滲透/安全測試策略
* 跨多個渠道重複使用內容
* 考慮以雲為重點的注意事項，例如：自動擴展、複製、資產管理、內部和托管服務之間的影響和差異、雲管理器、包管理

>[!NOTE]
>
>您的認證有效期為2年，從您通過認證之日起。 我們建議盡可能採用較新版本，因為它涵蓋較新的產品功能。

**考試詳細資訊：**

* 級別：主（3-5年經驗）
* 通過分數：29/50
* 時間：100分鐘
* 交貨：線上監控（需要攝像頭訪問）或test中心監控
* 可用語言：英語、日語
* 成本：225美元（全球）/150美元（印度）
* 考試ID:AD0-E117

**考試目標和範圍**

第1節：發現(18%)

* 將高級別業務目標轉化為功能需求
* 在假設情況下，評估體系結構的當前狀態
* 確定解決方案設計的非功能性技術要求

第二節：解決方案設計(44%)

* 給定業務需求、設計詳細的體系結構和解決方案
* 給定一個方案，為設計確定適當的安全解決方案
* 應用過程性概念將整合要求納入解決方案設計
* 確定解決方案中應包括哪些類型的效能和測試要求
* 應用過程性概念來設計可擴展和彈性的體系結構
* 在情況下，建議遷移策略

第三節：執行(22%)

* 給定一個方案，確定並解決在實施階段發現的設計問題
* 在設想的情況下，根據要求建議實施辦法
* 確定用於原型解決方案以驗證概念的適當方法
* 給定方案，診斷並解決實施過程中遇到的問題

第四節：維護(16%)

* 給定一個方案，診斷並解決生產環境中遇到的問題
* 應用程式概念規劃新功能和升級的使用
* 給定一個方案，針對非功能性需求優化解決方案
